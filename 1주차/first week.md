# 1주차 학습내용

## MARKDOWN 사용법 정리

- ### 제목 및 글머리

    ~~~
    # 제목1(주제목)
    ## 제목2(부제목)
    ### 제목3
    #### 제목4
    ##### 제목5
    ###### 제목6
    ~~~

- ### CodeBlock

    ~~~~Language
    ~~~Language
    Main Code
    ~~~
    ~~~~

- ### 수평선

    ~~~
    --------------------------
    ~~~

- ### Link(1. [Google](http:www.google.com) 2. 구글 : <www.google.com>)

    ~~~
    1. [홈페이지명](연결주소)
    2. 홈페이지명 : <연결주소>
    ~~~

    

- ### Emphasize

    ~~~
    기울기 : *sentence*
    굵게 : **sentence**
    밑줄 : <u>sentence</u>
    ~~~
